# most-fan-landing
A landing page for the most.fan startup

## Stack
* Vue 3 (composition api)
* TypeScript
* SCSS

## Project setup
node v14.10.0
yarn v1.22.15
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
